# Unit Testing - Zadania

# Zadanie 1
Przygotuj testy jednostkowe dla wybranych metod z klasy `com.example.Calculator`. Użyj:
- odpowiednich adnotacji
    - `@BeforeEach`
    - `@AfterEach`
    - `@BeforeAll`
    - `@AfterAll`
    - `@Test`
- API asercji
    - `@assertThrow`
    - `@assertAll`
    - `@assertTimeout`
- `@DisplayName`
- `@Nested`
- `@ParametrizedTest`
    - `@ValueSource`
    - `@CsvSource`
    - `@CsvFileSource`
    - `@MethodSource`
- On/Off
    - `@Disabled`
    - `@EnabledOnJre`
    - `@EnabledOnOs`
- `@Tag`
